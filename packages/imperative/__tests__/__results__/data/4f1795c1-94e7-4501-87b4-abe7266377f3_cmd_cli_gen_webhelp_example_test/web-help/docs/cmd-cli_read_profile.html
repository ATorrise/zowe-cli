Read some profiles

Usage

cmd-cli read profile [options]

Profile Options

  • --insecure-profile | --insecure-p (string)

    • The name of a (insecure) profile to load for this command execution.

  • --base-profile | --base-p (string)

    • The name of a (base) profile to load for this command execution.

Options

  • --amount | -a (number)

    • The amount of fruits.

  • --price | -p (number)

    • The price of one fruit.

  • --host (string)

    • Fruit host

  • --port (number)

    • Fruit port

  • --user (string)

    • Fruit username

  • --password (string)

    • Fruit password

  • --token-type (string)

    • Fruit token type

  • --token-value (string)

    • Fruit token value

  • --cert-file (local file path)

    • Fruit certificate file

  • --cert-key-file (local file path)

    • Fruit certificate key file
