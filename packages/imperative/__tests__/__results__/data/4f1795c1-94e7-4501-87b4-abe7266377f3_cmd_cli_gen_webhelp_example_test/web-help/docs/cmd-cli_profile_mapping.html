Tests Imperative's profile to CLI mapping capabilities.

Usage

cmd-cli profile mapping [options]

Required Options

  • --color | -c (string)

    • The color of the banana.

  • --banana-description | --bd (string)

    • A description of the banana

  • --mold-type | --mt (string)

    • The type of mold on the banana if any

  • --sweetness | -s (string)

    • The sweetness of the banana

      Default value: mild

Options

  • --ripe | -r (boolean)

    • Is the banana ripe?

  • --sides (number)

    • Number of sides on the banana

  • --names (array)

    • The names that this banana is known by

  • --amount | -a (number)

    • The amount of fruits.

  • --price | -p (number)

    • The price of one fruit.

  • --host (string)

    • Fruit host

  • --port (number)

    • Fruit port

  • --user (string)

    • Fruit username

  • --password (string)

    • Fruit password

  • --token-type (string)

    • Fruit token type

  • --token-value (string)

    • Fruit token value

  • --cert-file (local file path)

    • Fruit certificate file

  • --cert-key-file (local file path)

    • Fruit certificate key file

Profile Options

  • --banana-profile | --banana-p (string)

    • The name of a (banana) profile to load for this command execution.

  • --base-profile | --base-p (string)

    • The name of a (base) profile to load for this command execution.
