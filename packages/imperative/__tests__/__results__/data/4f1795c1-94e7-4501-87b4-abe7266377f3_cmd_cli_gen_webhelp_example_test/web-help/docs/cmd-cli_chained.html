chained handler test commands

Usage

cmd-cli chained <command>

Where <command> is one of the following:

Commands

  • fail-response-mapping - try to map a non-existent response field, not optional - fail
  • handler-chain-failure - a handler mid-chain throws an error
  • many-handlers - many handlers with argument mappings
  • mapping-notexist - try to map a non-existent response field, optional - succeed
  • optional-response-mapping - try to map a non-existent response field, optional - succeed
  • print-animals - print three animal names
  • print-cheetah - three handlers, but the first two are silent. only prints 'cheetah'
