Responds with error, headers, messages, data, etc. (all handler response APIs, except progress)

Usage

cmd-cli respond with-mixed-responses [options]

Required Options

  • --data-object | --da (json)

    • An well-formed JSON object to place in the command response.

  • --message-for-response | --mfr (string)

    • The message that will be placed in the output object.

  • --error-format-string | --efs (string)

    • A format string that will be used to create a message

  • --error-format-values | --efv (array)

    • The set of values to use in the format string.

  • --log-format-string | --lfs (string)

    • A format string that will be used to create a message

  • --log-format-values | --lfv (array)

    • The set of values to use in the format string.
