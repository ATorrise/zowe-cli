Invoke commands to validate that mapping profile fields to options is working correctly

Usage

cmd-cli profile <command>

Where <command> is one of the following:

Commands

  • mapping - Tests Imperative's profile to CLI mapping capabilities.
  • mapping-base - Tests handling of options across base and service profiles.
  • mapping-name-type - Tests that --type and --name are not filled in from profiles.
  • mapping-positional | mp - Tests Imperative's profile to CLI mapping capabilities.
