Kiwi Profile

Warning: This command has been deprecated.
Recommended replacement: The 'config set' command

Usage

cmd-cli profiles update kiwi-profile <profileName> [options]

Positional Arguments

  • profileName (string)

    • Specifies the name of the new kiwi profile. You can load this profile by using the name on commands that support the "--kiwi-profile" option.

Options

  • --amount | -a (number)

    • The amount of kiwis.

  • --price | -p (number)

    • The price of one kiwi.
