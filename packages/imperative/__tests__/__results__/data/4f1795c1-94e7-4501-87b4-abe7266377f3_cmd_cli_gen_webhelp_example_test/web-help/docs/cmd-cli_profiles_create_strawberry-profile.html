Strawberry Profile

Warning: This command has been deprecated.
Recommended replacement: The 'config init' command

Usage

cmd-cli profiles create strawberry-profile <profileName> [options]

Positional Arguments

  • profileName (string)

    • Specifies the name of the new strawberry profile. You can load this profile by using the name on commands that support the "--strawberry-profile" option.

Required Options

  • --amount | -a (number)

    • The amount of strawberries.

Options

  • --overwrite | --ow (boolean)

    • Overwrite the strawberry profile when a profile of the same name exists.

  • --disable-defaults | --dd (boolean)

    • Disable populating profile values of undefined properties with default values.
