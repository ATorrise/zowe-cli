cmd-cli ‚Ä∫ respond

Invoke various types of handlers that will formulate different responses via console messages, data objects, etc.

Usage

cmd-cli respond <command>

Where <command> is one of the following:

Commands

  • with-data-array - Responds with a string array
  • with-data-object - Responds with a JSON object
  • with-error-headers - Responds with error headers to the terminal/console
  • with-error-messages - Responds with error messages to the terminal/console
  • with-log-messages - Responds with log messages to the terminal/console
  • with-mixed-responses - Responds with error, headers, messages, etc
  • with-syntax-errors - Responds with syntax errors
  • with-wrapped-text - Responds with wrapped text and non wrapped text
