Log in to fruit authentication service

Usage

cmd-cli auth login fruit [options]

Options

  • --show-token | --st (boolean)

    • Show the token when login is successful. If specified, does not save the token to a profile.

  • --amount | -a (number)

    • The amount of fruits.

  • --price | -p (number)

    • The price of one fruit.

  • --host (string)

    • Fruit host

  • --port (number)

    • Fruit port

  • --user (string)

    • Fruit username

  • --password (string)

    • Fruit password

  • --cert-file (local file path)

    • Fruit certificate file

  • --cert-key-file (local file path)

    • Fruit certificate key file

Profile Options

  • --base-profile | --base-p (string)

    • The name of a (base) profile to load for this command execution.
