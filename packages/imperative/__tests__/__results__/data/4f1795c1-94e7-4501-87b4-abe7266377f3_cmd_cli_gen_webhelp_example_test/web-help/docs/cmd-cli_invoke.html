The invoke command allows you to invoke handlers and other test commands.

Usage

cmd-cli invoke <command>

Where <command> is one of the following:

Commands

  • exit-143 - Test handler that exits with status code 143
  • test-async-handler - Test basic async command handler completion
  • test-handler - Test handler with promise
  • unexpected-exception-handler - Test an unexpected exception
